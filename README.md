## Installation


```bash
 git clone https://github.com/nirvair09/mern-note-app.git
```

Get inside the server folder and run
```bash
cd server
npm install
nodemon index.js
```

Get inside the client folder and run
```bash
cd client
npm install
npm run dev
```
